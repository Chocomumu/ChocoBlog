## 分组查询
### 语法
```sql
SELECT 分组函数(列名)[, 分组列名/表达式]
FROM 表名
[WHERE 原始表列条件]
GROUP BY 分组列名/表达式
[HAVING 分组后聚合函数条件]
[ORDER BY语句]
```
### 分组查询特点
1.分组查询的筛选分为两类  
||数据源|位置|关键字|
|-|-|-|-|
|分组前筛选|原始表|GROUP BY前|WHERE|
|分组后筛选|分组后的结果集|GROUP BY后|HAVING|

2.分组函数作为条件一定是放在HAVING语句中  
3.能用分组前筛选的，尽量使用分组前筛选，主要就是分组列相关的筛选  
4.GROUP BY和HAVING后都支持SELECT中定义的别名
5.GROUP BY后可以跟多个列，按多个列进行分组
